# Nand2Tetris

An implementation of Nand2Tetris (without the chapter 12).

## Directories
- 05 : Memory, CPU, and computer
- 06 : Assembler
- 08 : Virtual machine
- 11 : Jack compiler

## Environment
- Windows 11
- Go 1.17.3
- OpenJDK 17.0.1 (runtime for Nand2Tetris tools)
- Make for windows 3.81
